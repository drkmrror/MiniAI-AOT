# MiniAI-AOT
 A basic AI template website. Example from AOT characters.

# Requirements
Use XAMMP for viewing the website when you open that has php files:
https://www.apachefriends.org/tr/index.html
If you want to do the same ai website, get an api key from an api website like openai or gemini. You can use your own api key by pasting the key in your ".env" file.
